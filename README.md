# 🚗 BMW Clone  

![BMW Clone Banner](https://raw.githubusercontent.com/ronik-bajakke/BMWClone/main/banner.png)

A sleek and visually appealing **BMW homepage clone** built using **HTML and CSS**.  
This project replicates the design, layout, and modern aesthetic of BMW’s official website — showcasing front-end development and responsive design skills using **static technologies only**. 

---

## 🌐 Live Demo  

👉 **[Visit the Live Site](https://ronik-bajakke.github.io/BMWClone/)**  

---

## 🎯 Project Overview  

This project was created as a **front-end design practice** to strengthen skills in layout structuring, animation, and modern styling.  
It focuses on **replicating BMW’s design elements** such as hero banners, smooth transitions, and clean UI sections — all achieved **without JavaScript or frameworks**.


---

## 🧠 Features  


- ✅ Responsive and modern homepage layout  
- 🏎️ Hero section with luxury car display  
- 🎨 Clean navigation bar and footer  
- 🖼️ Interior car image animation using CSS keyframes  
- 💬 Interactive hover effects on UI components  
- ⚡ Built using semantic HTML and modular CSS  

---

## 🏗️ Built With  

- **HTML5** — for semantic structure and accessibility  
- **CSS3** — for styling, layout, transitions, and animations  
- **Font Awesome** — for navigation and footer icons  
- **Google Fonts** — *Montserrat* and *Bebas Neue* for modern typography

## 🚀 How to Run Locally  

1. **Clone the repository**
   ```bash
   git clone https://github.com/Soham-patil18/BMWClone.git
   ```  
2. Open the project folder  
   ```bash
   cd BMWCLone
   ```  
3. Open `index.html` in any web browser
